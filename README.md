# Easy Subscriptions

SaaS para controle de mensalidades, clientes, cobranças recorrentes simples, pagamentos manuais e inadimplência para profissionais autônomos e pequenos negócios.

## Stack

- Laravel 13
- Vue.js 3
- TypeScript
- MySQL
- Tailwind CSS
- Vite
- Inertia.js

## Objetivo

Permitir que profissionais e pequenos negócios controlem clientes recorrentes, planos de mensalidade, vencimentos, pagamentos e cobranças de forma simples.

## Funcionalidades previstas

- Autenticação
- Dashboard
- CRUD de clientes
- CRUD de planos de mensalidade
- Vínculo de cliente com plano
- Geração de cobranças recorrentes
- Baixa manual de pagamentos
- Controle de inadimplência
- Mensagens copiáveis para WhatsApp
- Relatórios básicos
- Painel admin
- CRUD de planos do SaaS
- Configurações globais
- SEO
- robots.txt
- sitemap.xml
