# Manual — Easy Subscriptions

## Visão geral

O Easy Subscriptions é um sistema para controle de mensalidades, clientes, cobranças recorrentes simples, pagamentos manuais e inadimplência.

## Clientes

Nesta área será possível cadastrar e gerenciar clientes recorrentes.

## Planos de mensalidade

Nesta área será possível cadastrar os planos ou pacotes oferecidos pelo usuário.

Cada plano de mensalidade poderá conter:

- Nome
- Valor
- Periodicidade
- Dia de vencimento
- Status ativo/inativo

## Vínculo do cliente com plano

Cada cliente poderá ser vinculado a um plano de mensalidade.

Esse vínculo permitirá gerar cobranças recorrentes.

## Cobranças

O sistema poderá gerar cobranças conforme o plano vinculado ao cliente.

Cada cobrança poderá conter:

- Cliente
- Plano
- Valor
- Data de vencimento
- Status
- Data de pagamento
- Observação

## Status da cobrança

Status previstos:

- Pendente
- Pago
- Atrasado
- Cancelado

## Pagamentos

O sistema permitirá baixa manual de pagamentos.

## Inadimplência

O sistema poderá listar clientes com cobranças vencidas ou pendentes.

## WhatsApp

O sistema poderá gerar mensagens copiáveis para WhatsApp, como:

- Lembrete de vencimento
- Cobrança em atraso
- Confirmação de pagamento

## Relatórios

Relatórios previstos:

- Cobranças a vencer
- Cobranças vencidas
- Cobranças pagas
- Receita prevista
- Receita recebida

## Planos do SaaS

O sistema terá plano gratuito e planos pagos configuráveis pelo painel admin.

## Painel admin

O admin poderá gerenciar usuários, planos do SaaS e configurações globais do sistema.

## Observação

Este manual será atualizado conforme o desenvolvimento evoluir.
